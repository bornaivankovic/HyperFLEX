Returns all VSDNs belonging to user (tenant, admin, ...). For each VSDN returns Hosts, Controller and logical links. Does NOT return any physical switches

Return type:JSON serialized Dictionary
get_physical_topo(user_type)[source]¶

Returns physical topology based on user. If user is a tenant, physical ressources allocated for tenant’s infrastructure are returned. If user is admin whole physical infrastructure (inluding hypervisor) is returned. Physical topology means switches and links between them.

Parameters:user (int) – Identifier of user (1 for admin)
Returns:topology – List of Dictionaries
Return type:JSON serialized List

Example

{

vsdn_id: <id>, nodes: [

{
‘id’: <id>, ‘type’: ‘switch’, ‘name’: <name>, ‘dpid’: <dpid>, ‘ip’: <ip>, ‘ip_port’: <ip_port>, ‘num_ports’: <num_ports>, ‘cplane’: <True/False>

] edges: [

{
‘from_node’: <id>, ‘from_port’: <port>, ‘to_node’: <id>, ‘to_port’: <id>

]

}
