intelligence.guihandling module¶

This module processes requests made by users through the gui. The main taks of this module is to orchestrate operations (adding logical link –> get embedding –> write embedding) for writing and retrieving of data

class intelligence.guihandling.ManagementGuiControllerHandler[source]¶

Handles persisting and triggering of processes related to GUI events. Uses intelligence module to identify embedding but handles persisting of changes into the db itself.
